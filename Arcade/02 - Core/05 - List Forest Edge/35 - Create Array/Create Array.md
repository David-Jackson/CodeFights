Given an integer `size`, return array of length `size` filled with `1`s.

**Example**

For `size = 4`, the output should be
`createArray(size) = [1, 1, 1, 1]`.

**Input/Output**

*   **[time limit] 4000ms (js)**

*   **[input] integer size**

    A positive integer.

    _Guaranteed constraints:_
    `1 ≤ size ≤ 10`.

*   **[output] array.integer**
