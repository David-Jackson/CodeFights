Find the sum of the angles of a convex _n-gon_ for a given `n`.

**Example**

For `n = 3`, the output should be
`sumOfTheAngles(n) = 180`.

![](images/example.png)

**Input/Output**

*   **[time limit] 4000ms (js)**

*   **[input] integer n**

    _Guaranteed constraints:_
    `3 ≤ n ≤ 15`.

*   **[output] integer**
