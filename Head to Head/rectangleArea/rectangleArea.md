Find the area of a rectangle with given sides.

**Example**

For `a = 2` and `b = 3`, the output should be
`rectangleArea(a, b) = 6`.

**Input/Output**

*   **[time limit] 4000ms (js)**

*   **[input] integer a**

    _Guaranteed constraints:_
    `1 ≤ a ≤ 20`.

*   **[input] integer b**

    _Guaranteed constraints:_
    `1 ≤ b ≤ 20`.

*   **[output] integer**
